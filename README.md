# Wavey
> Javascript & React/Redux frontend for Wavey, a music production app that uses motions to play sounds.

Backend Ruby on Rails API repo: https://github.com/kristinatong/wavey-backend

## Installation

OS X & Linux:

```sh
npm install
```
